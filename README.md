# Muhammad Hamza - Personal Portfolio

A modern, responsive, and professional personal portfolio built with **Next.js, Tailwind CSS, and Framer Motion**.  
Fully animated, graphics-rich, and mobile-friendly.

## Features

- **Hero Section:** Personal picture, name, and title.  
- **Projects Section:** Auto-generated project images with GitHub links.  
- **Pages:** Home, About Me, My Work, Contact (with working contact form).  
- Smooth transitions and Framer Motion animations.  

## Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Design:** Figma
